# API Usage

Base URL:
https://dog.ceo/api/

Endpoint:
breeds/image/random

Method:
GET

Example Response:
{
  "message": "https://images.dog.ceo/breeds/hound-afghan/n02088094_1003.jpg",
  "status": "success"
}

Usage in App:
The app sends a GET request and retrieves the image URL from the "message" field.
